Bulk amplitude manipulation of partials, by combining the amplitudes of two incoming partial sources.
Input A
The first partial source.
Input B
The second partial source.
Scale
A scalar value S.
Operation
The manipulation to perform:
+
A + B + S
-
max(A - B - B, 0)
x
A x B
inv x
A x (1 - B)
compress
1 - (1 - A) x (1 - B)
average
S x A + (1 - S) x B
min
min(A x (1 - S), B)
max
max(A, (1 - S), B)
filter
If B < S then 0 else A
filternot
If B >= S then 0 else A
fill
If A > S then A else B
threshold
If A > S then 1 else 0
scaledown
A x S + B, (Scales A in the range 0...1)
scaleup
A x (S + 1) + B (Scales A in the range 1...2)
scalefar
A x (S + 9) + B (Scales A in the range 1...10)
clampdown
min(A, S) + B
clampup
max(A, S) + B
Normalize
Normalizes the result after the operation.
