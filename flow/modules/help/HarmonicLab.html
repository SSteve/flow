A programmmable, modulatable harmonics generator.

Adds up to three kinds of waves. Each wave has a particular low pass filtering function and is constrained to a set of harmonics. If an earlier wave uses a given harmonic, it takes precedence over later waves.

Amt 1 ... 3
The strength of the filtering of the given wave.
Gain 1 ... 3
The amplitude of the partials in the given wave.
Type 1 ... 3
The filter shape. The shapes are given by their equation.
Constraint 1 ... 3
The harmonics to which the wave is constrained.
