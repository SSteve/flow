This module makes it possible to control a combobox or checkbox on some target module via modulation.
  1. Connect any output of the target module (either modulation or partials) into a Target input on the Choice module. Doing so informs Choice which target module you would like to manipulate.

  2. Next, choose the combobox or checkbox of interest via the Option dial.

  3. Finally, change or modulate the Value dial.
The Target Inputs
A modulation input and a partials input to connect to any output of your target module.
Option
The combobox or checkbox to manipulate.
Value
The setting of the combobox or checkbox.
